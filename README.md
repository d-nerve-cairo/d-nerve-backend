# D-Nerve Backend

FastAPI backend and PostgreSQL database for Cairo's informal microbus transit system.

## Features
- REST API with 7+ endpoints
- PostgreSQL + PostGIS for geospatial queries
- Real-time GPS data ingestion
- Route and ETA serving

## Tech Stack
- Python 3.11
- FastAPI
- PostgreSQL 15 + PostGIS
- SQLAlchemy

## Setup
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app/main.py
```

## API Documentation
Once running, visit: http://localhost:8000/docs
